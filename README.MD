# 🎯 Objectiu del repositori

Aquest repositori pretén:

* ajudar a **planificar el projecte**
* proporcionar **bones pràctiques**
* oferir **exemples de documentació**
* centralitzar el material de suport

---

# 👨‍💻 Desenvolupament del projecte

Els projectes han de ser **aplicacions web full-stack**, combinant frontend, backend i base de dades.

L'objectiu és aplicar els coneixements adquirits durant el cicle:

* desenvolupament web
* bases de dades
* APIs
* arquitectura d'aplicacions
* desplegament
